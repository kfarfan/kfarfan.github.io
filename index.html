Music Player

This is a music player similar to Spotify. The user is able to pick a song, skip around, mess with the volume knobs, and ultimately - enjoy their music! This was created using JavaScript and the jQuery library. Click the picture to listen to the Red Hot Chili Peppers!

Quiz App

Quiz app using the Reavealing Module pattern, made with pure JavaScript. I learned how to use the Revealing Module pattern to organize my code in a clean and clear way. It's important to have clean code that is organized, and modules help to accomplish that. Separation of concerns is the beauty of Object Oriented programming!

React News App

This is an app built in React using the Hacker News API. You can search and filter by keywords and have unlimited pages. I learned how to use props and state to render UI changes and how to call API's using React. I also use destructuring to clean up the code.

Guess the Color

This is a color guessing game that was built on jQuery. An array is filled with randomly generated colors and the user is able to click through them.

Calculator

This is a calculator built using HTML5, CSS3, and JavaScript. This shows DOM manipulation, and the use of module objects to render numbers on the screen.

Pomodoro

This was inspired from The Odin Project. I built this from scratch, and used an OOP approach. I created a class inside of a module with methods to increase or decrease the time depending on users actions. This shows state and dom manipulation, paired with an elegant design.
